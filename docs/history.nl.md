# Projectgeschiedenis

Taal: [English](https://tijnschouten.github.io/appie/history/) | **Nederlands**

Deze pagina bewaart de oorspronkelijke projectbrief waarmee `python-appie` is gestart.

Dat is nuttige context, omdat de live API tijdens de implementatie afweek van enkele oorspronkelijke aannames. Daardoor moesten meerdere endpoints en payloads tijdens het bouwen worden aangepast.

## Oorspronkelijke projectbrief

Zie de Engelse versie van deze pagina voor de volledige oorspronkelijke tekst. De kern is dat het project begon als een greenfield Python-package voor een onofficiële AH-client, met `uv`, `ruff`, `pytest`, async API-modules en een login-CLI.

Lees verder: [Ontwikkeling](development.md) voor lokale tooling, kwaliteitschecks en de contributieworkflow.
