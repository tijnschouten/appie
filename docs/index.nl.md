# python-appie

Taal: [English](https://tijnschouten.github.io/appie/) | **Nederlands**

`python-appie` is een onofficiële async Python-client voor de Albert Heijn API.

Ondersteunde onderdelen:

- browser-login via `appie-login`
- tokenopslag en automatische refresh
- product zoeken en productdetails ophalen
- bonoverzichten ophalen
- bonregels ophalen voor een specifieke bon
- item toevoegen aan de boodschappenlijst

## Huidige status

Deze package gebruikt een onofficiële API. Dat betekent:

- de API kan zonder waarschuwing veranderen
- velden en endpoints kunnen breken
- voorzichtig gebruik is verstandig
